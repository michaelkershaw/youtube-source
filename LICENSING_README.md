# Licensing

License management tools and documentation are located in the `licensing/` folder:

- `licensing/LICENSING.md` - Complete license system documentation
- `licensing/generate_key.cpp` - License key generator source code
- `licensing/build_keygen.bat` - Build script for the key generator
- `licensing/generate_key.exe` - Compiled key generator tool

## Quick Start

1. Go to the `licensing/` folder
2. Run `build_keygen.bat` to compile the generator
3. Run `generate_key.exe` to create and track license keys

See `licensing/LICENSING.md` for full documentation.
