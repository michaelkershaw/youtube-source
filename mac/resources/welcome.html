YouTube Source for VirtualDJ

Welcome to the YouTube Source plugin installer for VirtualDJ on macOS.

This installer will set up:

  • YouTube Source plugin — search, stream and download YouTube audio/video directly in VirtualDJ
  • yt-dlp — the YouTube extraction engine
  • ffmpeg — audio/video transcoding
  • Web UI — the plugin's modern browser-based interface

Requirements:

  • VirtualDJ 2021 or later (64-bit)
  • macOS 11.0 (Big Sur) or later
  • An active internet connection for YouTube streaming

Please close VirtualDJ before continuing if it is currently running.
